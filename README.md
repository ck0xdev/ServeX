# ServeX

A modern web services platform built with React, Tailwind CSS, and Firebase.

## Features

- 🔐 Firebase Authentication (Email/Password)
- 👤 User Dashboard with Profile & Messages
- 📁 File Upload System (Base64)
- 🛡️ Admin Panel for Management
- 💎 Neumorphic UI Design
- 📱 Fully Responsive

## Tech Stack

- React 18
- Tailwind CSS 3
- Firebase (Auth, Firestore)
- Vite
- Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
